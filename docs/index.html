Deploy in 5 minutes

Follow these simple steps to get started.

1

The Telegram Bot

Message @BotFather on Telegram. Run /newbot and follow the prompts to get your Bot Token.

2

Firebase Database

1. Create a project in the Firebase Console.

2. Enable Firestore Database.

3. Go to Project Settings > Service Accounts and Generate a new private key (JSON).

3

GitHub Configuration

In your GitHub repository, go to Settings > Secrets and variables > Actions and add these secrets:

TELEGRAM_BOT_TOKEN
GEMINI_API_KEY
FIREBASE_SERVICE_ACCOUNT (Full JSON content)
4

Push to Deploy

Simply push this code to your repository. The GitHub Action will trigger and start listening for your commands!

git add . && git commit -m "Setup ClosedAI" && git push
